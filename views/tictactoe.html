TicTacToe API

Try it out!

curl https://api.abhinavpola.com/tictactoe/:boardstring

RESPONSE
{ boardString: "user input", evaluation: "Evaluation.result" }

Generate SVG of board

https://api.abhinavpola.com/tictactoe/boardview/:boardstring

RESPONSE
A 3x3 grid representation of the current board state given a valid input string.

How to save SVG

Copy and paste the code within SVG tags into a text editor and save with .svg extension.
